Crate finchers_session[−][src]

Session support for Finchers.

Supported backends:

  • Cookie
  • In-memory database
  • Redis (requires the feature flag feature = "redis")

Feature Flags

  • redis - enable Redis backend (default: off)
  • secure - enable signing and encryption support for Cookie values (default: on. it adds the crate ring to dependencies).
